Описание технических решений

Для данного объекта была применена весьма интересная и рабочая схема вентиляции и хорошего кондиционирования воздуха. Как правило для помещений всегда есть ограничение по електричеству. И данный объект не стал исключением. За счёт вытяжной системы мы решаем вопрос с удалением запахов и отработанного воздуха. А за счёт достаточного разряжения воздуха мы подтягиваем эго в систему кондиционирования где он будучи отфильтрованным подогревается в кондиционере. Такое решение помогает Нашим клиентам сэкономить на капитальных затратах и решить одним решением две задачи: свежий воздух и прохлада.

Описание оборудования

Для данного объекта использовались центробежные вентиляторы Вентс ВКМц которые вытягивают отработанный воздух.

В качестве фильтров использовались карманные фильтры с классом очистки G4 (для крупных частиц пыли. Для регулирования подачи воздуха используют дроссельный заслонки с сервоприводом.

Для кондиционирования воздуха были применении канальные сплит системы с частичным подмесом свежего воздуха.
